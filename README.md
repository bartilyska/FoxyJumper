# FoxyJumper
Made with my friends VV01T3K, korneliuszw.

Controls:

- WSAD/Arrows - move
- Space - jump ; skip dialog
- F - pickup granade
- Q - drop grenade
- LMB - throw grenade
- ESC - pause menu
